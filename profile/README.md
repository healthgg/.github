### 👩‍🏫김한솔

#### UI
  - 페이지 : 홈, 검색, 게시글 작성, 게시글 수정, 게시글 상세, 채팅방
  - 공통 컴포넌트 : 게시글 템플릿, 버튼
#### 기능
  - 유저 검색, 게시글 등록 및 수정, 게시글 상세 확인, 댓글 등록, 팔로워 게시글 불러오기, 좋아요 기능
    
### 👨‍🏫김용식

#### API 구현
- 검색 : 운동기구 및 영양소 목록 검색 기능 구현
- 무한스크롤 : 운동 기구 및 영양소 목록 무한스크롤 구현
#### 홈서버 구축
- 클라우드 서비스를 사용하지 않고 호스트 PC에 홈서버 구축
- VMware를 사용하여 개발 배포서버 구축
